# 🚀 FinTwin AI

> **Your Financial Digital Twin for Smarter Financial Decisions**

FinTwin AI is an AI-powered Financial Intelligence Platform that creates a **Financial Digital Twin** for every user. Instead of functioning as a traditional expense tracker, it builds a personalized financial model to analyze financial health, provide intelligent insights, simulate financial scenarios, and power AI-driven financial decision-making.

This project is being developed as a production-quality flagship portfolio project with a focus on software engineering, machine learning, retrieval-augmented generation (RAG), and modern SaaS design.

---

## ✨ Current Features

### 🔐 Authentication

* Secure user registration
* User login & logout
* Password hashing using bcrypt
* Session management
* PostgreSQL-backed authentication

### 👤 Financial Digital Twin

Every user receives a personalized Financial Digital Twin that stores:

* Monthly Income
* Emergency Fund
* Investments
* Loans
* Insurance Cover
* Monthly Budget
* Risk Appetite
* Investment Style
* Preferred Currency

### 📊 Financial Health Engine

Automatically calculates:

* Financial Health Score
* Monthly Savings
* Savings Rate
* Debt Ratio
* Investment Ratio
* Emergency Fund Coverage

### 🤖 AI Financial Insights

Rule-based intelligent financial recommendations such as:

* Increase monthly savings
* Build emergency fund
* Reduce debt
* Improve investment allocation

### 📈 Premium Dashboard

* Financial Health Gauge
* KPI Cards
* Financial Ratios
* AI Insights
* Quick Actions
